# CreASHions by Asha — website preview

A preview of the **CreASHionsbyAsha** website — handmade, hand-poured resin art
(tumblers, earrings, coasters, pens & decor) from Cypress, Texas.

This is an early, clickable design preview so Asha can **see and feel** the site before it
goes fully live. Browse the shop, open a product, read the story, and try the cart.

## 👀 See it live

**→ The live link is in the “About” section of this repository (top-right), or visit the
GitHub Pages URL for this repo.**

The shop itself lives at:
`design-system/ui_kits/website/index.html`

## What’s here so far
- A boutique storefront — Home, Shop, Collections, Product, My Story, and a Custom-order page.
- Asha’s real products, themes, and brand (the watercolor dragonfly look, her colors).
- A no-code **Studio** editor for managing products and text:
  `design-system/ui_kits/website/admin.html`
- The full brand/design system (colors, type, components) under `design-system/`.

## What’s coming next (not in this preview yet)
- Real, secure checkout (cards + Apple Pay + Google Pay) — payments are in **demo mode** here.
- Customer accounts / sign-in (Google, Microsoft, Facebook, Apple, or email).
- One-click editing that publishes changes to the live site automatically.
- A real web address (e.g. `creashionsbyasha.com`) and search-engine setup.

## A note for Asha
Nothing here can take real payments yet — the “Checkout” button shows a friendly demo
message. This is purely to show how your shop will **look and work**. Tell us what you’d
love to change — colors, words, which products lead — and we’ll shape it around you. 💛
